# Project Summary

DOM XSS ML is an AI-based graduation project focused on detecting DOM-Based XSS using structural analysis of web pages.

The system collects accessible web pages, extracts DOM features, applies trained machine learning models, and generates prediction results with general mitigation recommendations.

## Main Flow

1. URL input
2. Page crawling and filtering
3. DOM parsing and cleaning
4. Feature extraction
5. AI prediction
6. Results and report generation

## Models

The project compares XGBoost, AdaBoost, LightGBM, Decision Tree, Random Forest, and MLP.
