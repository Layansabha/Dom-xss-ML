# Repository Structure

```text
Dom-xss-ML/
├── README.md
├── .gitignore
├── docs/
│   ├── project-summary.md
│   └── structure.md
├── src/
│   └── pipeline/
├── data/
├── models/
└── reports/
```

## Folder Purpose

- `src/`: source code and implementation files.
- `src/pipeline/`: crawling, DOM extraction, feature extraction, prediction, and reporting logic.
- `data/`: datasets and extracted DOM features.
- `models/`: trained models and model comparison files.
- `reports/`: generated reports, evaluation outputs, and screenshots.
- `docs/`: project explanation and documentation.
